# AI-Generated Frontend Portfolio  
### Author: Aditya Pawar

This is a modern, fully-responsive frontend portfolio web application generated with the help of AI. It demonstrates clean UI, reusable components, animations, user interaction, and a scalable file structure.

The application is built using:

- **React**  
- **Vite**  
- **Tailwind CSS**  
- **Framer Motion** (for smooth animations)  
- **Lucide Icons**

The goal of this project is to showcase frontend skills, design clarity, component architecture, and UI-focused development.
